# Documents juridiques

Placez vos documents PDF originaux dans ce dossier :

- `avertissement.pdf` - Avertissement disciplinaire
- `contestation.pdf` - Lettre de contestation
- `lettre-licenciement.pdf` - Lettre de licenciement
- `conclusions.pdf` - Conclusions aux prud'hommes
- `decision-justice.pdf` - Décision du conseil de prud'hommes

Puis référencez-les dans vos articles Markdown :

```markdown
[Télécharger l'avertissement](/docs/avertissement.pdf)
```

Les fichiers dans `static/` sont servis tels quels à la racine du site.
